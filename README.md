# CIC
CIC is a program that creates cool images.


# Environment

* Mac OS
* Python 3.6.1
* pillow
* pyqt5

# Setup
    $ pip install pyqt5
    $ pip install Pillow

# Usage
    $ python3 CIC.py
    and just open the image and convert
